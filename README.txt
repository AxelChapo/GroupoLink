Download the project / Télécharger le projet

How to install this project:

In command prompt:

Step 1: go to the "front" directory
Step 2: type "npm install"
Step 3: type "ng serve"

Step 4: go to "API" directory
Step 5: type "npm install"
Step 6: type "nodemon server"
